<p><i>PCAPdroid-mitm</i> is a <a href='https://github.com/emanuele-f/PCAPdroid' target='_blank' rel='nofollow noopener'>PCAPdroid</a> addon which uses <a href='https://mitmproxy.org/' target='_blank' rel='nofollow noopener'>mitmproxy</a> to decrypt the TLS/SSL connections and show the decrypted data in the app. The addon uses the open source framework <a href='https://chaquo.com/chaquopy' target='_blank' rel='nofollow noopener'>chaquopy</a> to bundle and run python modules. The native python modules are pre-built and installed from the chaquopy <a href='https://chaquo.com/pypi-7.0' target='_blank' rel='nofollow noopener'>pip repository</a>.</p>
